# Blog

Jon Snow's blog.

## Installation

Check out the latest version of Blog and enter it

```bash
$ git clone https://github.com/air-examples/blog.git
$ cd blog
```

and run it

```bash
$ go run main.go
```

then visit `http://localhost:8080`.
